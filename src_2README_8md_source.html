3 Some of these directories come from entire outside repositories brought in
4 using [git-subtree][]. This means that the source files are inserted directly
5 into the `rippled` repository. They can be edited and committed just as if they
6 were normal files.
7 [git-subtree]: https://github.com/apenwarr/git-subtree
8 
9 If you create a commit that contains files both from a subtree, and from the
10 `rippled` source tree, please use care when designing the commit message, since
11 it will appear in the subtree's individual repository when the changes are
12 pushed back to the upstream. Better yet, do not mix files from subtrees and
13 `ripple` in the same commit at all.
